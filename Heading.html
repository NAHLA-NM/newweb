Heading Tag Demo in HTML

This page shows how <h1> to <h6> tags work in HTML.

1. HTML Code Display:

<h1>This is Heading 1</h1>
<h2>This is Heading 2</h2>
<h3>This is Heading 3</h3>
<h4>This is Heading 4</h4>
<h5>This is Heading 5</h5>
<h6>This is Heading 6</h6>

2. Rendered Output:

This is Heading 1

This is Heading 2

This is Heading 3

This is Heading 4

This is Heading 5
This is Heading 6

Note: The heading size decreases from h1 (largest) to h6 (smallest).
